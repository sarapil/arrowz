# دليل منصة Arrowz للاتصالات الموحدة

## نظرة عامة

منصة Arrowz للاتصالات الموحدة (Omni-Channel) توفر واجهة موحدة للتواصل مع العملاء عبر قنوات متعددة:

- **واتساب** (عبر Meta Cloud API أو الخادم الخاص)
- **تيليجرام** (عبر Bot API)
- **البريد الإلكتروني** (مدمج مع Frappe)
- **الهاتف/PBX** (عبر Arrowz VoIP)
- **اجتماعات الفيديو** (عبر OpenMeetings)

## التثبيت

### المتطلبات

- Frappe Framework v15+
- ERPNext (موصى به)
- تطبيق frappe_whatsapp (للواتساب)
- تطبيق frappe_telegram (للتيليجرام)

### خطوات التثبيت

```bash
# تثبيت تطبيق Arrowz
bench get-app --branch main arrowz

# تثبيت على الموقع
bench --site your-site.localhost install-app arrowz

# تشغيل الترحيل
bench --site your-site.localhost migrate

# بناء الأصول
bench build --app arrowz
```

## الإعداد

### 1. إعداد الواتساب

#### Meta Cloud API (الطريقة الموصى بها)

1. اذهب إلى قائمة **AZ Omni Provider**
2. أنشئ مزود جديد:
   - **نوع المزود**: WhatsApp Cloud
   - **اسم المزود**: Meta Cloud WhatsApp
   - **رابط الخدمة**: `https://graph.facebook.com/v18.0`
   - **مفتاح API**: رمز الوصول من Meta
   - **سر التطبيق**: سر تطبيق Meta
   - **رمز التحقق**: رمز مخصص للتحقق من Webhook

3. أنشئ **AZ Omni Channel**:
   - **نوع القناة**: WhatsApp
   - **المزود**: اختر المزود الذي أنشأته
   - **معرف القناة**: معرف رقم الهاتف من Meta
   - **اسم القناة**: واتساب الأعمال

4. إعداد Webhook في Meta Developer Console:
   ```
   الرابط: https://your-site.com/api/method/arrowz.api.webhooks.whatsapp_cloud_webhook
   رمز التحقق: (نفس المعد في المزود)
   ```

### 2. إعداد تيليجرام

1. أنشئ بوت تيليجرام عبر @BotFather
2. أنشئ **AZ Omni Provider**:
   - **نوع المزود**: Telegram
   - **مفتاح API**: رمز البوت

3. أنشئ **AZ Omni Channel** للبوت

4. تسجيل Webhook:
   ```python
   frappe.call({
       method: "arrowz.api.webhooks.register_telegram_webhook",
       args: {
           bot_token: "YOUR_BOT_TOKEN"
       }
   })
   ```

### 3. إعداد OpenMeetings

1. ثبت وأعد خادم OpenMeetings
2. أنشئ **AZ Omni Provider**:
   - **نوع المزود**: OpenMeetings
   - **رابط الخدمة**: `https://your-om-server.com/openmeetings`
   - **اسم المستخدم**: اسم المشرف
   - **مفتاح API**: كلمة مرور المشرف

## دليل المستخدم

### عرض سجل الاتصالات

1. افتح أي مستند مدعوم (عميل محتمل، عميل، إلخ)
2. ستظهر لوحة الاتصالات أسفل حقول النموذج
3. انقر على تبويبات القنوات للفلترة
4. انقر على محادثة لعرض التفاصيل

### إرسال رسالة واتساب

1. انقر على زر واتساب في الإجراءات السريعة
2. أدخل رقم الهاتف (يُملأ تلقائياً إن وُجد)
3. إذا انتهت نافذة 24 ساعة، اختر قالباً
4. اكتب رسالتك
5. انقر إرسال

### جدولة اجتماع فيديو

1. انقر على زر الفيديو في الإجراءات السريعة
2. أدخل اسم الاجتماع والوقت
3. أضف المشاركين بعناوين بريدهم
4. انقر جدولة
5. تُرسل الدعوات تلقائياً

### إدارة المحادثات

- **تعيين**: تعيين المحادثات لأعضاء الفريق
- **إغلاق**: إغلاق المحادثات المحلولة
- **الردود السريعة**: استخدام قوالب الرد السريع

## دليل المدير

### إدارة المزودين

المزودون هم إعدادات الاتصال لكل خدمة.

**حقول AZ Omni Provider:**
- نوع المزود
- اسم المزود
- رابط الخدمة
- بيانات المصادقة
- حدود المعدل
- القدرات

### إدارة القنوات

القنوات هي نقاط الاتصال الفردية (أرقام الهواتف، البوتات).

**حقول AZ Omni Channel:**
- نوع القناة
- المزود (رابط)
- معرف القناة
- قواعد التوجيه
- ساعات العمل
- رسائل الرد التلقائي

### المراقبة

1. **حالة Webhook**: تحقق من `/api/method/arrowz.api.webhooks.get_webhook_status`
2. **المحادثات النشطة**: عرض قائمة AZ Conversation Session
3. **غرف الاجتماعات**: عرض قائمة AZ Meeting Room
4. **سجلات الأخطاء**: تحقق من Error Log للمشاكل

### المهام المجدولة

تعمل المهام التالية تلقائياً:

| المهمة | الجدول | الوصف |
|--------|--------|-------|
| check_window_expiry | كل 15 دقيقة | تحقق من انتهاء نافذة 24 ساعة للواتساب |
| sync_openmeetings_status | كل ساعة | مزامنة حالة الاجتماعات |
| cleanup_ended_conversations | يومياً | أرشفة المحادثات القديمة |
| cleanup_temporary_rooms | يومياً | حذف الغرف المؤقتة القديمة |
| generate_omni_channel_report | أسبوعياً | إرسال تقرير التحليلات |

## الأسئلة الشائعة

### لماذا لا تصل رسائل الواتساب؟

1. تحقق من إعداد Webhook في Meta Developer Console
2. تأكد من أن رابط Webhook يعمل (HTTPS مطلوب)
3. تحقق من سجل الأخطاء
4. تأكد من تطابق رمز التحقق

### لماذا لا يستجيب بوت تيليجرام؟

1. تأكد من صحة رمز البوت
2. تحقق من تسجيل Webhook: `getWebhookInfo` API
3. تأكد من أن HTTPS يعمل مع شهادة صالحة

### ماذا تعني "نافذة 24 ساعة"؟

واتساب يسمح بإرسال رسائل حرة فقط خلال 24 ساعة من آخر رسالة من العميل. بعد ذلك، يجب استخدام قوالب معتمدة.

### كيف أضيف قناة اتصال جديدة؟

1. أنشئ Driver جديد في `arrowz/integrations/`
2. أضف نوع المزود في AZ Omni Provider
3. أضف Webhook endpoint
4. سجل في hooks.py
