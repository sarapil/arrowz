# [Announcement] Arrowz — Unified Communications for ERPNext | Open Source

Hi Frappe Community! 👋

We're excited to share **Arrowz**, a new open-source communication app for Frappe/ERPNext.

## What it does

✅ WebRTC Browser Softphone
✅ WhatsApp Cloud API Integration
✅ Telegram Bot Messaging
✅ OpenMeetings Video Conferencing
✅ MikroTik Network Management
✅ FreePBX/Asterisk AMI Integration
✅ Automatic Call Logging to CRM
✅ Multi-site Network Topology

## Why we built it

- Disconnected communication tools costing $1000+/month
- Lost customer conversations across platforms
- No call records in CRM/ERP
- Network issues taking hours to diagnose

We couldn't find a good communication solution that integrates natively with ERPNext, so we built one.

## Tech Stack

- **Backend:** Python, Frappe Framework v16
- **Frontend:** JavaScript, Frappe UI, frappe_visual components
- **Database:** MariaDB (standard Frappe)
- **License:** MIT
- **Dependencies:** frappe_visual, caps, arkan_help

## Installation

```bash
bench get-app https://github.com/sarapil/arrowz
bench --site your-site install-app arrowz
bench --site your-site migrate
```
