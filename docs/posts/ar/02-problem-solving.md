## يعمل مع ERPNext الحالي لديك

Arrowz لا يستبدل ERPNext — بل **يُوسّعه**. ثبّته بجانب إعدادك الحالي:

```bash
bench get-app https://github.com/sarapil/arrowz
bench --site your-site install-app arrowz
```

كل بياناتك وسير عملك وتخصيصاتك الحالية تبقى كما هي.
