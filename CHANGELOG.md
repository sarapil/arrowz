# Changelog — Arrowz

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [16.0.0] — 2026-04-04

### Added
- Enterprise VoIP & Omni-Channel Communications platform
- WebRTC Softphone (JsSIP-based) with navbar integration
- WhatsApp Cloud API + Telegram Bot omni-channel support
- OpenMeetings video conferencing integration
- FreePBX/Asterisk AMI call management
- MikroTik RouterOS full network management (11 modules)
- Device Provider Layer — abstract pattern for Linux Box + MikroTik
- Local PBX Monitor for debugging (`/mnt/pbx` volume)
- SQL injection protections on all API endpoints
- CI/CD workflows (ci.yml, linters.yml, release.yml)
- About page (`/arrowz-about`) and onboarding page (`/arrowz-onboarding`)
- Arabic + 10 language translations (923 strings)
- File-based contextual help (`help/en/`, `help/ar/`)
- Desktop Icon workspace grouping (9 workspaces)
- CAPS integration for capability-based access control
