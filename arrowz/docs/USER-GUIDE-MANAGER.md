## 📺 لوحة المراقبة الحية (Wallboard)

### الوصول:
**Arrowz** ← **Manager Wallboard**

### المكونات الرئيسية:

#### 1. بطاقات المؤشرات الرئيسية (KPIs)
```
┌─────────────┬─────────────┬─────────────┬─────────────┐
│ المكالمات   │ في الانتظار │ متوسط       │ مستوى       │
│ النشطة     │             │ الانتظار    │ الخدمة      │
│    12      │     3       │   0:45      │   87%       │
└─────────────┴─────────────┴─────────────┴─────────────┘
```

| المؤشر | الوصف |
|--------|-------|
| المكالمات النشطة | عدد المكالمات الجارية حالياً |
| في الانتظار | المكالمات المنتظرة في الطابور |
| متوسط الانتظار | متوسط وقت انتظار المتصلين |
| مستوى الخدمة | نسبة المكالمات المُجابة خلال الوقت المستهدف |

#### 2. خريطة الوكلاء
```
┌──────────────────────────────────────────────────────┐
│                    حالة الوكلاء                      │
├──────────┬──────────┬──────────┬──────────┬─────────┤
│ 🟢 أحمد  │ 🟡 سارة  │ 🔴 محمد  │ 🟢 نورة  │ ⚪ علي  │
│ في مكالمة│ استراحة  │ غير متوفر │ متاح    │ غير متصل│
│ 2:34    │ 5:00    │          │         │         │
└──────────┴──────────┴──────────┴──────────┴─────────┘
```

**ألوان الحالة:**
- 🟢 **أخضر** - متاح أو في مكالمة
- 🟡 **أصفر** - استراحة أو أعمال إدارية
- 🔴 **أحمر** - غير متوفر
- ⚪ **رمادي** - غير متصل

#### 3. قائمة المكالمات النشطة
```
┌──────────────────────────────────────────────────────┐
│ الوكيل    │ المتصل      │ الاتجاه │ المدة  │ إجراءات │
├───────────┼─────────────┼─────────┼────────┼─────────┤
│ أحمد      │ 0501234567  │ وارد   │ 2:34  │ 👂 🎧   │
│ نورة      │ 0559876543  │ صادر   │ 1:15  │ 👂 🎧   │
└──────────────────────────────────────────────────────┘
```

**الإجراءات المتاحة:**
- 👂 **الاستماع** - استمع للمكالمة بدون علم الطرفين
- 🎧 **المشاركة** - انضم للمكالمة (ثلاثي)

## 👥 إدارة الوكلاء

### تغيير حالة وكيل:
1. انقر على اسم الوكيل في Wallboard
2. اختر **تغيير الحالة**
3. حدد الحالة الجديدة

### إجبار الخروج:
1. انقر على اسم الوكيل
2. اختر **إجبار الخروج**
3. أكد الإجراء

### مراقبة الأداء الفوري:
1. انقر على اسم الوكيل
2. اختر **عرض التفاصيل**
3. راجع:
   - المكالمات اليوم
   - متوسط المدة
   - وقت التوفر
   - المكالمات الفائتة

---

## 🎧 مراقبة المكالمات

### الاستماع الصامت (Silent Monitoring):
1. في قائمة المكالمات النشطة، انقر 👂
2. ستستمع للمكالمة بدون علم الطرفين
3. للخروج، انقر على **إنهاء الاستماع**

### المشاركة في المكالمة (Barge-In):
1. انقر على 🎧 بجوار المكالمة
2. ستنضم للمكالمة كطرف ثالث
3. يمكنك التحدث مع الطرفين

### الهمس (Whisper):
1. انقر على 🗣️ بجوار المكالمة
2. ستتحدث مع الوكيل فقط
3. لن يسمعك المتصل

## 📈 تقارير الأداء

### التقرير اليومي:
- يُرسل تلقائياً عبر البريد
- يشمل ملخص المكالمات
- مقارنة بمتوسط الأسبوع

### تقارير مخصصة:
1. اذهب إلى **Arrowz** ← **Reports**
2. اختر نوع التقرير
3. حدد النطاق الزمني
4. انقر **إنشاء**

### أنواع التقارير:
| التقرير | الوصف |
|---------|-------|
| ملخص المكالمات | إحصائيات عامة |
| أداء الوكلاء | تفاصيل لكل وكيل |
| الطوابير | أداء كل طابور |
| SLA | مستوى الخدمة |
| تحليل الأوقات | ساعات الذروة |

## ⚙️ إعدادات الطوابير

### الوصول:
**Arrowz** ← **إعدادات الطوابير**

### الإعدادات المتاحة:
- **وقت الانتظار الأقصى** - قبل التحويل لبريد صوتي
- **رسالة الانتظار** - ما يسمعه المتصل
- **أولوية التوزيع** - كيفية توزيع المكالمات
- **مستوى الخدمة المستهدف** - النسبة المطلوبة

---

## 🔔 التنبيهات

### إعداد التنبيهات:
1. اذهب إلى **الإعدادات** ← **التنبيهات**
2. اختر نوع التنبيه:
   - وقت انتظار طويل
   - مكالمات فائتة كثيرة
   - وكيل غير متوفر
3. حدد الحد الأقصى
4. اختر طريقة الإشعار (بريد، إشعار، SMS)

### التنبيهات التلقائية:
- ⚠️ وقت انتظار > 3 دقائق
- 🔴 وكيل غير متوفر > 15 دقيقة
- 📉 مستوى خدمة < 80%
